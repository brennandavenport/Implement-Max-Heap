# Implement-Max-Heap

Implement a `MaxHeap` in Python.

## Rules

- Do not use built-in heap libraries such as `heapq`.
- Only use the internet for Python syntax reference.
- Work with a partner.

## Your Task

1. Clone the repository.
2. Create your branch:

```sh
git checkout -b feature/your_name-partner_name
```

3. Go to the [implementation/](/implementation/) folder.
4. Create a new file named `your_name-partner_name.py`.
5. Copy the starter code from [implementation/example.py](/implementation/example.py) into your new file.
6. Implement the `push` and `pop` methods in your `MaxHeap` class.
7. Run the tests, after all tests pass please show either Brennan or Adithya


## Project Files

- [base_heap.py](/base_heap.py): Abstract base class with the required heap interface.
- [implementation/example.py](/implementation/example.py): Starter template for your solution.
- [engine.py](/engine.py): Runs all tests against your heap implementation.


## How To Run Tests

Run the engine and pass it the path to your implementation file:

```sh
python3 engine.py implementation/your_name-partner_name.py
```

You need all test to pass to complete this problem
